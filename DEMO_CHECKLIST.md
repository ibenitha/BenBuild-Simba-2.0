## 🚨 Emergency Procedures

### If Internet Fails

1. Switch to mobile hotspot
2. Or show demo video
3. Or walk through with screenshots

### If Application Crashes

1. Restart development server
2. Or switch to backup laptop
3. Or continue with explanation + screenshots

### If Feature Doesn't Work

1. Stay calm
2. Acknowledge the issue
3. Explain what should happen
4. Move to next feature
5. Come back if time permits

### If You Forget Something

1. Take a breath
2. Refer to notes (if allowed)
3. Continue with confidence
4. Don't apologize excessively
